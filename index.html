Operator HMI — interactive prototype

A simplified click-through of the touchscreen the operator uses at the station: pick the model, set count & bin, scan, confirm the label. Step through scans one at a time, or auto-run the rest of the current bin's count hands-free — the run always includes one deliberately wrong scan so you can see exactly how a bad part is caught.

HOW TO USE THIS

  • Tap a model, set target qty & bin, then start.
  • Tap SIMULATE SCAN to count parts one at a time, or AUTO-RUN REMAINING to fill the rest of this bin's count hands-free — its progress (e.g. 7/12) is the exact same count shown in the ring, never a different number.
  • Every auto-run includes exactly one wrong-part scan (at a random point) so you can see the catch — plus a small dashed button to test one wrong scan on demand.
Rule: one part number per bin, strictly enforced. A wrong-part scan is rejected without resetting the count. Auto-run only fills this bin's count — applying the printed label to the bin stays a manual, deliberate step.
